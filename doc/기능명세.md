ContainerList 기능 :
- ContainerList.tsx - API 연결 및 호스트 선택 기능 추가
- ContainerList.css - 호스트 선택 드롭다운 스타일 추가
- 추가된 기능:

- 호스트 목록을 /api/hosts에서 가져옴
- 호스트 선택 드롭다운 추가
- 선택된 호스트의 컨테이너 목록을 /api/ps2/{host}에서 가져옴
- 호스트 변경 시 자동으로 컨테이너 목록 갱신
- Refresh 버튼으로 수동 새로고침
- 에러 메시지 및 빈 상태 표시
- All / Running / Exited 필터 버튼
- 각 버튼에 해당 상태의 컨테이너 개수 표시
- 필터링된 결과 개수 표시 (예: "Showing 3 of 10 containers")
- 필터링 결과가 없을 때 상태에 맞는 메시지 표시

- startContainer 함수: POST /api/start2 API 호출
- Start 버튼 클릭 시 컨테이너 시작
- 시작 중일 때 버튼 텍스트가 "Starting..."으로 변경
- 시작 중인 컨테이너의 버튼 비활성화
- 성공 시 컨테이너 목록 자동 새로고침
- 에러 발생 시 에러 메시지 표시

- stopContainer 함수: POST /api/stop2 API 호출
- Stop 버튼 클릭 시 컨테이너 중지
- 중지 중일 때 버튼 텍스트가 "Stopping..."으로 변경
- 액션 진행 중인 컨테이너의 버튼 비활성화
- 성공 시 컨테이너 목록 자동 새로고침
- 에러 발생 시 에러 메시지 표시

- Inspect 버튼 연결

Inspect 레이아웃 구성:
- Basic Information: ID, Name, Image, Platform, Created, Restart Count
- State: Status, Running, Paused, Exit Code, Started At, Finished At
- Configuration: Hostname, User, Working Dir, Command, Entrypoint, Env, Labels
- Network: IP Address, Gateway, MAC Address, Port Mappings, Networks
- Mounts: Type, Source, Destination, RW/RO
